# KIT_TAXI_SHARE

Made by Kaqui (2019/07/07)

## 이번 프로젝트의 컨셉은 금오공대 택시 쉐어입니다 !

### 동기
- 많은 학우들이 대나무 숲, 에브리타임, 단톡방에서 매번 귀찮게 택시를 같이 탈 사람을 찾는 것을 목격
- 여러 글들이 올라오기 때문에 내가 올린 글이 묻히는 경향 다분


### 목표
- 금오공대 학생을 위해 택시 쉐어 전용 어플 작성
- 실제 배포 목적
- 기획~개발~유지보수 단계 경험
- Front & Back & DB & Doc 작성 경험

### 구현 기능
- 택시 공유글 CRUD
- 마이페이지
- 관심글 알람 설정
- 댓글
- Email 인증
- 비밀번호 암호화(PBE)
- 각방 채팅(**Socket**)

### 사용하는 것들
- Kotlin
- NodeJS
- Git
- Notion(개인 기록용)
- MVP
- **Socket**
- Socket.io
